# European Job Market Visualizer

An interactive treemap exploring **436 ISCO-08 occupations** across the **27 EU member states**, with AI exposure scoring, multilingual support, and real Eurostat employment data.

> Forked from [karpathy/jobs](https://github.com/karpathy/jobs) and fully adapted for the European labor market.

🔗 **Live demo**: [alexandrucruceanu.github.io/EU-jobs](https://alexandrucruceanu.github.io/EU-jobs/)

---

## Features

| Feature | Details |
|:--|:--|
| **436 occupations** | Full ISCO-08 Level 4 (Unit Groups) from the ESCO taxonomy |
| **198M total jobs** | Real EU27 2023 Eurostat census employment figures |
| **19 industry sectors** | Domain-grouped treemap (Healthcare, Technology, Construction, etc.) |
| **AI exposure scores** | 0–10 scale per occupation, generated by Gemini 3.1 Flash-Lite |
| **24 languages** | All official EU languages, with ESCO-backed native job titles |
| **4 color layers** | Growth Outlook · Median Pay · Education · AI Exposure |
| **Real-time search** | Filter by job title or 4-digit ISCO code |
| **Methodology disclaimer** | Transparent about imputed vs. exact data |

## Data Pipeline

```
scrape_eu.py → process_eu.py → make_csv_eu.py → score.py → build_site_data.py
                                                              ↓
                                                         site/data.json
```

| Step | Script | Output |
|:--:|:--|:--|
| 1 | `scrape_eu.py` | Download ESCO taxonomy JSON into `raw_eu/` |
| 2 | `process_eu.py` | Parse into clean Markdown (`pages_eu/`) |
| 3 | `make_csv_eu.py` | Build structured CSV with EU-adapted stats |
| 4 | `score.py` | Generate AI exposure scores via Gemini API |
| 5 | `build_site_data.py` | Merge CSV + scores → `site/data.json` |
| 6 | `translate_ui.py` | Translate UI into 23 additional EU languages |
| 7 | `inject_native_titles.py` | Inject official ESCO job titles per language |

## Quick Start

```bash
# Clone
git clone https://github.com/alexandrucruceanu/EU-jobs.git
cd EU-jobs

# Set up API key
cp .env.example .env
# Edit .env and add your Gemini API key (https://aistudio.google.com/apikey)

# Install dependencies
pip install google-genai python-dotenv requests

# Rebuild from scratch (optional — data files are already included)
python scrape_eu.py
python process_eu.py
python make_csv_eu.py
python score.py
python build_site_data.py

# Serve locally
python -m http.server 8000 --directory site
```

## Key Files

| File | Description |
|:--|:--|
| `occupations_eu.csv` | Structured stats: pay, education, employment, growth |
| `scores_eu.json` | AI exposure scores (0–10) with rationales |
| `site/data.json` | Frontend payload (auto-generated) |
| `site/index.html` | Interactive treemap visualization |
| `site/i18n/*.json` | 24 language dictionaries |
| `prompt.md` | Full dataset summary for LLM analysis |
