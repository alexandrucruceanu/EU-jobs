# European Job Market Visualizer

An interactive treemap exploring **436 ISCO-08 occupations** across the **27 EU member states**, with AI exposure scoring, multilingual support, and real Eurostat employment data.

> Forked from [karpathy/jobs](https://github.com/karpathy/jobs) and fully adapted for the European labor market.

## Features

| Feature | Details |
|:--|:--|
| **436 occupations** | Full ISCO-08 Level 4 (Unit Groups) from the ESCO taxonomy |
| **27 EU member states** | Per-country data with real Eurostat employment figures |
| **Interactive map view** | Choropleth map of Europe with 4 metric layers |
| **19 industry sectors** | Domain-grouped treemap (Healthcare, Technology, Construction, etc.) |
| **AI exposure scores** | 0â€“10 scale per occupation, generated by Gemini 3.1 Flash-Lite |
| **24 languages** | All official EU languages, with ESCO-backed native job titles |
| **4 color layers** | Growth Outlook Â· Median Pay Â· Education Â· AI Exposure |
| **Country-specific pay** | Wage levels scaled per country using Eurostat indices |

| **Real-time search** | Filter by job title or 4-digit ISCO code |

## Quick Start

```bash
# Clone
git clone https://github.com/alexandrucruceanu/EU-jobs.git
cd EU-jobs

# Set up API key
cp .env.example .env
# Edit .env and add your Gemini API key (https://aistudio.google.com/apikey)

# Install dependencies
pip install google-genai python-dotenv requests

# Rebuild from scratch (optional â€” data files are already included)
python scrape_eu.py
python process_eu.py
python make_csv_eu.py
python score.py
python build_site_data.py

# Serve locally
python -m http.server 8000 --directory site
```
